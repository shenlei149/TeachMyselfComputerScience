首先，我们来看一下手算乘法。
```
        1000
        1001
        ----
        1000
       0000
      0000
     1000
     -------
     1001000
```
第一个操作数称为被乘数（`multiplicand`），第二个操作数称为乘数（`multiplier`），最后结果是乘积（`product`）。

乘积可能比被乘数和乘数都要打。不考虑符号位的情况下，如果被乘数有 $n$ 比特，乘数有 $m$ 比特，那么总共需要 $n+m$ 来存储乘积。由于通常情况下，我们使用 32 比特来存储两个 32 比特数的乘积，因此类似于加法，需要考虑溢出的情况。
